# **Crud** - WebApi 2.0 - AngularJs
## Projeto consiste em 2 projetos separados, Front-end e Back-end. Crud com layout responsivo.
> 1. Banck-end: WebApi C# (IDE: Visual studio 2017- .Net Framework 4.6.1), arquitetura DDD.
> 2. Front-end: AngularJs (IDE : Visual Studio Code), CSS Bootstrap 4.0.0-beta.

- Obs: métodos e serviço não contem as melhores metodologias para nomenclatura de serviços. Ex: [exemplo1](https://www.vinaysahni.com/best-practices-for-a-pragmatic-restful-api), [exemplo2](https://blog.mwaysolutions.com/2014/06/05/10-best-practices-for-better-restful-api/)
